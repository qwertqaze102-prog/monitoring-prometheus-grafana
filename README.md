# Monitoring: Prometheus + Grafana

**Stack skills:** `Prometheus` · `Grafana` · `Docker` · `Python` · `Linux`

Works on Windows Docker Desktop (no host /proc mounts).

## Run
```bash
docker compose up -d --build
```
- Prometheus: http://127.0.0.1:9090
- Grafana: http://127.0.0.1:3000 (admin/admin)
- Dashboard folder: Lab → Runnable Lab Overview
